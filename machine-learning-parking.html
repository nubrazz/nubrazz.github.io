MACHINE LEARNING WITH UNITY3D: PARKING

  • UnityMachine Learning

Machine Learning With Unity3D is a project where I used Unity's ML-Agents to run a machine learning simulation. This project aims to deepen my understanding of the Unity game engine and also satisfy my interest in machine learning.


Goal

The main goal of the project is to create an AI that can play my own game. The game is relatively simple, with the AI controlling a car driver trying to park the car without crashing in the process.


Challenges

There are many challenges in this project, such as designing a reward system that effectively motivates the AI to improve its performance. The biggest challenge is integrating machine learning into my game, specifically figuring out how to make the AI control the car. These questions and challenges made me excited to work on this project, and I am looking forward to learning more about how AI can be implemented in games.


Results

In the end, this project yielded very positive results. First and foremost, the AI was able to play the game quite well, which is a great achievement. Additionally, I was able to improve my knowledge of machine learning, and gained a deeper understanding of how to implement it in my games. Overall, I am very pleased with the outcome of this project and the new skills and knowledge that I have acquired.
