MACHINE LEARNING WITH UNITY3D: PATH FINDING

  • UnityMachine Learning

Machine Learning With Unity3D is a project where I used Unity's ML-Agents to run a machine learning simulation. This project aims to deepen my understanding of the Unity game engine and also satisfy my interest in machine learning.


Goal

The goal of the project is to create an AI that can navigate to a checkpoint without hitting obstacles and barriers.
It is a simple and straightforward goal.


Challenges

As this is my first time using Unity's ML-Agents, I encountered several challenges. The main challenges for this project were not so much about how to make the AI reach the checkpoint, but rather how to properly use ML-Agents to start a machine learning simulation. My lack of knowledge about Unity's ML-Agents made this project more challenging, but it also sparked my interest in overcoming these challenges.


Results

Despite the challenges, I was able to successfully teach the AI how to navigate to the checkpoint. In addition, I gained a deeper understanding of Unity's ML-Agents and the concept of machine learning.
Overall, this was an extremely valuable learning experience.
